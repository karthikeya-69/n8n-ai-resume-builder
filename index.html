Features

What it does

Built to be end-to-end — no manual steps, no copy-pasting between tools.

🤖
Gemini AI generation
Prompts Gemini to write structured, ATS-optimised resume content based on the user's raw input.
📄
Automatic PDF creation
Resume content is passed to PDF.co API and returned as a print-ready, downloadable document.
📧
Gmail automation
The finished PDF is attached and delivered to the user's inbox without any manual step.
⚡
Webhook-triggered
Any form tool, frontend, or API can fire the workflow — no n8n UI interaction required after setup.
🎯
ATS-friendly format
Output is structured for keyword scanning by applicant tracking systems — no tables, clean hierarchy.
🧩
Modular nodes
Swap Gemini for another model, or Gmail for SendGrid, without touching the rest of the pipeline.
